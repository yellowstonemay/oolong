Accelerometer

USING THE SAMPLE
To run this sample, choose Project > Set Active SDK > Device and connect a device. Then click Build and Go.

NOTE: You cannot run this sample in the simulator because it requires OpenGL ES, which is not implemented for the simulator.

BUILD REQUIREMENTS
Mac OS X 10.5.2, Xcode 3.1, iPhone OS 2.0

RUNTIME REQUIREMENTS
Mac OS X 10.5.2, iPhone OS 2.0
This sample application runs only on a device, not on the simulator.
